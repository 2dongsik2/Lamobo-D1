[Python]
To run python on D1, please follow these steps:
1. Compile with the command './xcompiler.sh' and generate all the output files to 'install' folder.
2. Copy 'install' folder and script 'setenv_python.sh' to sdcard.
3. Start D1 with sdcard, and execute commands to enter python:
$ . /mnt/setenv_python.sh
$ python
